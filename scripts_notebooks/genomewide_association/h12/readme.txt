To calculate delta h12, you need the permutation file generated by [./genomewide_association/fst/llineup_random_permuation.r]. 
Using the bash script [./h12/h12_randomisation.sh], we load the python script h12.py. The bash script loops the H12.py functions through each population and in all the contigs to generate H12.tsv file.
We use the calculate_h12_window_p_value.r script to calculate delta H12 and visualise in plots.
